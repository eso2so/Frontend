font-family

  • font-family는 폰트를 지정하는 속성이다.
  • 여러 개의 폰트를 지정하면 우선순위에 따라 적용된다.
  • 폰트 이름에 띄어쓰기가 있다면 ""(큰따옴표) 안에 넣는다.

font-weight

  • font-weight는 글씨의 가중도(굵기)를 지정한다.
  • 100 ~ 900까지 100 단위로 지정한다. 혹은 bold, light와 같은 키워드로 지정 가능하다. 단, 해당 가중도의 폰트가 있어야 한다. 없으면 가장 가까운 최대 숫자를 지정한다.

font-size

  • font-size는 글씨의 크기를 지정한다.
  • px과 같은 고정 단위 혹은 em, rem과 같은 상대 단위를 사용한다.

color

  • color는 글씨의 색상을 지정한다.
  • a태그는 기본 컬러가 있어 상위 요소로부터 상속받지 않는다.

text-align

  • text-align은 글씨의 정렬 방법을 지정한다.
  • 영역을 나누는 블록 요소에 사용한다.
  • Lorem, ipsum dolor sit amet consectetur adipisicing elit. Repudiandae, provident similique vel id temporibus officia impedit tempore voluptas repellat ad, consectetur totam dolores iusto alias quasi blanditiis sed ratione culpa.

text-transform

  • text-transform은 글씨를 대문자, 소문자로 변형시킨다.

text-decoration

  • 글씨에 줄을 그어서 꾸민다.
  • 선의 위치, 모양, 색상, 두께를 지정할 수 있다.

line-height / word-spacing, letter-spacing

  • line-height는 줄 간격을 조정한다.
  • Lorem ipsum dolor sit, amet consectetur adipisicing elit. Distinctio laudantium possimus numquam libero. Id obcaecati eligendi quis libero doloribus, soluta animi velit similique sed aspernatur minima eveniet repudiandae! Quisquam, quaerat?
  • word-spacing은 단어간의 간격, letter-spacing은 글자간의 간격을 조정한다.
